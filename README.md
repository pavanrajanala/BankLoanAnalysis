# 🏦 Bank Loan Analysis (Python Project)

## 📌 About This Project

This project is all about understanding how banks decide whether to approve or reject a loan. I used Python with Pandas, NumPy, and Matplotlib to explore real loan data and find useful patterns.

Basically, the goal is to answer:

👉 Why do some people get loans approved while others don’t?

---

## ❓ What I Tried to Find

In this project, I focused on questions like:

* How many loans are approved vs rejected?
* Does income affect loan approval?
* Does credit history really matter?
* Which areas (urban/rural) get more approvals?
* What is the average loan amount people apply for?

---

## 🧰 Tools I Used

* Python 🐍
* Pandas (for handling data)
* NumPy (for calculations)
* Matplotlib (for charts)
* Google Colab (to run everything)

---

## 📂 Dataset Info

The dataset has details about loan applicants like:

* Gender, marital status, education
* Income details
* Loan amount & term
* Credit history
* Property area
* Loan status (Approved / Rejected)

---

## 🔍 What I Did in This Project

### 1. Loaded the Data

Opened the dataset and checked how it looks.

### 2. Cleaned It

Fixed missing values and made the data ready for analysis.

### 3. Explored the Data

Looked at patterns like:

* Who gets loans more often
* How income affects approval
* How credit history changes results

---

## 📊 What I Found (Insights)

* People with good credit history have much higher chances of getting a loan.
* Higher income usually helps in approval.
* Urban and semi-urban areas get more approvals.
* Loan amount and repayment details also matter a lot.
